# 50 Заданий по типам данных в Dart

## Блок 1: Integer (int) — Задания 1–10

### 1. Основные операции с целыми числами
Напишите программу, которая:
- Создаёт две целые переменные (a = 15, b = 4)
- Выводит их сумму, разность, произведение
- Выводит целочисленное деление и остаток

**Ожидаемый результат:**
```
Сумма: 19
Разность: 11
Произведение: 60
Целочисленное деление: 3
Остаток: 3
```

### 5. Факториал (простая версия)

### 6. Проверка диапазона

### 7. Сумма цифр

### 8. Количество цифр

### 9. Палиндром из цифр

## Подсказки для решения

### Для заданий 1–10 (int):
- Используйте операторы: +, -, *, ~/, %
- Используйте методы: .abs(), .isEven, .isOdd
- Для факториала используйте цикл или рекурсию

### Для заданий 11–20 (double):
- Используйте методы: .ceil(), .floor(), .round(), .toInt()
- Помните о точности double (не всегда 100%)
- Для математических операций используйте math библиотеку при необходимости

### Для заданий 21–30 (String):
- Интерполяция: $variable и ${expression}
- Методы: .length, .toUpperCase(), .toLowerCase(), .contains(), .indexOf()
- Подстрока: .substring(start, end)
- Разбиение: .split(separator)

### Для заданий 31–35 (bool):
- Операции: && (AND), || (OR), ! (NOT)
- Сравнения: >, <, >=, <=, ==, !=
- Тернарный оператор: condition ? trueValue : falseValue

### Для заданий 36–40 (List):
- Методы: .add(), .remove(), .length
- Фильтрация: .where((element) => condition).toList()
- Преобразование: .map((element) => transformation).toList()
- Сортировка: .sort()

### Для заданий 41–45 (Map):
- Доступ: map[key], map['key'] ?? defaultValue
- Итерация: .forEach((key, value) => ...)
- Методы: .keys, .values, .entries

### Для заданий 46–50 (Set):
- Операции: .union(), .intersection(), .difference()
- Добавление: .add(), .addAll()
- Проверка: .contains()
